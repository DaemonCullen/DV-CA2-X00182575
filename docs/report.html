Bootstrap

For my CA I used many different features of bootstrap to help create smooth and beautiful website. I used the Carousel to help show information without to much being on the screen at once, allowing the user to scroll through the information. To help the user navigate I used Offcanvas to make a quick and easy navigation side bar for the user. I also used the Cards feature to help quickly make nice repeatable sections. For my contact me section I used both the Form and the Alert to help make an imersive feeling contact form. I also used bootstraps great Icons to allow me to add simple icons quick. Finally, throught the entire page I used the long list of smaller features bootstrap has to help personalise the site more such as: col-md, p, m, g, justify-content-center, row, shadow-lg and many more.
